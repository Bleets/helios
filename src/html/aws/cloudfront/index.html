Cloudfront

  • Display all node about cloudfront 
    MATCH (n:CLOUDFRONT) RETURN n UNION MATCH (n:CLOUDFRONT_ORIGIN_CUSTOM) RETURN n UNION MATCH (n:CLOUDFRONT_ORIGIN_S3) RETURN n
  • Display important CLOUDFRONT_ORIGIN_CUSTOM (Detect single point of failure) 
    MATCH (a:CLOUDFRONT)-[:ORIGIN|:DEFAULT_BEHAVIOR]->(b:CLOUDFRONT_ORIGIN_CUSTOM) RETURN b, COLLECT(a) as cloudfront ORDER BY SIZE(cloudfront) DESC LIMIT 10
    Info : Change the value of LIMIT if you want more or less precision
